Mac OSX 终端走代理

2019-07-17 #Mac OS

用此方法可以加速各种依赖安装的速度

以 zsh 作为说明

➜ ~ vim ~/.zshrc

添加如下代理配置:

# proxy list
alias proxy='export all_proxy=socks5://127.0.0.1:1080'
alias unproxy='unset all_proxy'

:wq 保存退出

➜  ~ source ~/.zshrc

使用 proxy 前先查看下当前的ip地址：

➜  ~ curl ip.cn
当前 IP：112.64.xxx.xx 来自：上海市 联通

没问题，终端走了代理，brew update 顺畅了- -

如果不需要走代理，执行：

➜  ~ unproxy   
➜  ~ curl ip.cn
当前 IP：112.64.xxx.xx 来自：上海市 联通
